# 📜 Code of Conduct

## 🌍 Our Commitment

PGT Global Network – GSSoC Open Source Edition is committed to creating a welcoming, respectful, and inclusive environment for everyone.

We believe open source should be a space for learning, collaboration, and meaningful contribution.

---

# 🤝 Expected Behavior

All contributors, maintainers, and participants are expected to:

- Be respectful and professional
- Use welcoming and inclusive language
- Accept constructive feedback gracefully
- Focus on project improvement
- Support fellow contributors
- Communicate clearly and honestly
- Respect different perspectives and experiences

---

# ❌ Unacceptable Behavior

The following behaviors are not tolerated:

- Harassment or discrimination of any kind
- Personal attacks or insulting language
- Trolling or disruptive behavior
- Spam or self-promotion unrelated to project goals
- Sharing private information without permission
- Plagiarism or intentionally harmful code contributions

---

# ⚖️ Enforcement Responsibilities

Project admins are responsible for maintaining community standards and may:

- Warn contributors
- Request behavior correction
- Reject pull requests
- Remove contributors from participation if necessary

---

# 📢 Reporting Issues

If you experience or witness unacceptable behavior, please report it through official project communication channels or GitHub issue moderation.

All reports will be reviewed fairly and confidentially.

---

# 🚀 Our Standard

We are building more than code — we are building a purpose-driven ecosystem through technology, leadership, and collaboration.

By participating, you agree to uphold these standards.

---

### Thank you for contributing responsibly to PGT Global Network.
